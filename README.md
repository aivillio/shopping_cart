# Shopping Cart Management System (Python + MySQL)

This project is a **command-line based Shopping Cart Management System** built using **Python** and **MySQL**.  
It allows users to log in, add items to a cart, update or delete items, view the cart, calculate total price with discounts, and complete payment.

---

## Features

- User login (User ID, Username, Pincode)
- Add items to cart with quantity
- Display cart contents
- Update cart items and quantity
- Delete items from cart
- Calculate total payable amount (with discount)
- Payment simulation
- MySQL database integration
